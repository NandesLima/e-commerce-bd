# 🛒 E-commerce Database Modeling & Analytics

Este repositório contém a modelagem completa de um banco de dados para e-commerce, incluindo o processo de Engenharia de Requisitos, diagramas e os scripts SQL de criação (DDL) e consultas (DQL).

## 🚀 Estrutura do Repositório

- **`database/model/`**: Contém o arquivo original de modelagem lógica (`e-commerce.mwb`) para ser aberto no MySQL Workbench.
- **`database/scripts/`**: Contém o script `schema.sql` (geração das tabelas e chaves) e `queries.sql` (consultas de negócio).
- **`docs/`**: Código-fonte da documentação oficial do projeto em MkDocs.
